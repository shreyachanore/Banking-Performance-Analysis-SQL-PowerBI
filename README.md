# Banking-Performance-Analysis-SQL-PowerBI

##  Project Overview
This project presents an end-to-end Banking Performance Analysis using SQL and Power BI.  
The objective is to analyze customer behavior, account performance, loan distribution, and overall banking operations to generate actionable business insights.

The project follows a complete data workflow:
Data Extraction (SQL) → Data Cleaning → Data Modeling → Dashboard Development (Power BI) → Business Insights

---

##  Tools & Technologies Used
- SQL (Data extraction & transformation)
- Power BI (Dashboard & visualization)
- DAX (Calculated measures & KPIs)
- Excel (Data preprocessing)

---

##  Key KPIs & Metrics
- Total Customers: 100
- Active Accounts: 50
- Inactive Accounts: 50
- Total Balance: 2.63M
- Average Customer Age: 45.26
- Average Interest Rate: 8.89%
- Credit Utilization Ratio: 0.50

---

##  Dashboard Insights

### 1️⃣ Account Analysis
- Savings accounts hold the highest total balance (~1M).
- Investment accounts are the second highest.
- 50% accounts are inactive — strong re-engagement opportunity.

### 2️⃣ Loan Analysis
- Personal Loans lead in total loan amount (~2M).
- Auto Loans have the highest interest rate (9.86%).
- Mortgage loans maintain moderate balance and interest.

### 3️⃣ Customer Demographics
- Female customers: 61%
- Male customers: 39%
- Majority customers fall in 51–65 age group.

### 4️⃣ Transaction Insights
- Steady monthly transaction growth observed.
- Credit utilization shows positive correlation with balance.

### 5️⃣ Branch & Manager Performance
- Balanced customer distribution across branches.
- Performance comparison done based on balance contribution.
